# Portfólio — Chayene de Souza

Este é o repositório do meu portfólio pessoal, desenvolvido com HTML, CSS e JavaScript, com foco em responsividade, experiência do usuário e apresentação profissional de projetos web.

O projeto foi pensado para funcionar bem em diferentes dispositivos (mobile, tablet e desktop), destacando meus trabalhos, serviços e formas de contato de maneira clara e acessível.

## Tecnologias utilizadas
HTML5, CSS3 e JavaScript.

## Funcionalidades
Layout responsivo, navegação intuitiva, menu mobile, seções de projetos, serviços e contato, além de integração com links externos.
